Get started in 4 steps

1

Install the module

Install AI Database Chatbot from Apps. Depends on base, web, mail — included in every Odoo install.
2

Open AI Configuration

Go to AI Assistant > Configuration > AI Configuration and create a new config record.
3

Choose a provider & paste your API key

Select OpenAI, Anthropic, Gemini, DeepSeek, or Perplexity. Paste your key, choose a model, click Test Connection.
4

Start chatting

Click the floating chat button on any backend page and ask things like "Show me open invoices from Azure Interior".
