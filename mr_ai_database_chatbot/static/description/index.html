Get started in 4 steps

1

Install the module

Install AI Database Chatbot from Apps. The module depends on base, web, and mail — all included in every Odoo installation.

2

Open AI Configuration

Go to AI Assistant → Configuration → AI Configuration and create a new config record.

3

Choose a provider & paste your API key

Select OpenAI, Anthropic, Gemini, DeepSeek, or Perplexity. Paste your API key, choose a model, and click Test Connection.

4

Start chatting

Click the floating chat button in the bottom-right corner of any backend page and start asking questions like "Show me open invoices from Azure Interior" or "What is the total revenue this month?"
